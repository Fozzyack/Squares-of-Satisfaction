# Squares of Satisfaction

A visual habit tracker that helps you build better habits through GitHub-style contribution squares.

## About

Squares of Satisfaction is a habit tracking application that visualizes your progress using a grid of squares, similar to GitHub's contribution graph. Track any habit - drinking water, coding, sleeping, exercise, or anything else - and watch your consistency grow.

## Features

- Visual habit tracking with square-based grid interface
- Track multiple habits simultaneously
- Clean, simple interface
- Built with Next.js and Go

## Getting Started

### Frontend (Client)

```bash
cd client
bun install
bun run dev
```

The client will be available at `http://localhost:3000`

### Backend

```bash
cd backend
go run main.go
```

## Tech Stack

- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS
- **Backend**: Go
